# PROJECT CONTEXT

## Product Vision

The Product North Star for Version 1 is:

> "Automatically create the best possible annual family photo album."

Version 1 is intentionally focused on this single objective to maximize quality, speed of execution, and product clarity.

Future album types are already part of the long-term vision but are intentionally postponed, including:

- Vacation Albums
- Gift Albums
- Event Albums
- Story Timeline

The software architecture should remain extensible so these future album types can be added without major redesign.
